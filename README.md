# homebrew-claude-usage

Homebrew tap for [Claude Usage](https://github.com/Sven1603/claude-usage) — a
macOS menu bar app showing live Claude 5-hour session usage.

## Install

    brew install --cask sven1603/claude-usage/claude-usage

Or:

    brew tap sven1603/claude-usage
    brew install --cask claude-usage

The app is an unsigned hobby build; the cask clears the Gatekeeper quarantine
flag automatically on install, so no manual step is needed.
